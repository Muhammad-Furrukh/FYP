# Previously committed:
10: add  x3, x1, x2   → tag 20 (committed)
11: sub  x5, x3, x4   → tag 25 (committed)

# In ROB, not yet committed:
12: mul  x3, x5, x6   → tag 42 (executing, not ready)
13: xor  x7, x3, x8   → tag 45 (waiting for tag 42)

# Just renamed this cycle:
14: and  x5, x7, x9   → tag 53 (just issued, not ready)
15: or   x10, x5, x11 → tag 54 (just issued, not ready)
```

## Lookup Operation Example

**Instruction 16 arrives:** `add x12, x3, x5`
```
┌─────────────────────────────────────────────────────────────┐
│ LOOKUP 0: x3 (rs1)                                          │
├─────────────────────────────────────────────────────────────┤
│ Step 1: specTag[3] = 0b0101010 (tag 42)                    │
│ Step 2: Extract MSB = 0 (normal tag)                        │
│ Step 3: tagAvail[42] = 0 (not ready)                        │
│ Step 4: Check writeback... no match                         │
│ Step 5: Check same-cycle issue... no match                  │
│                                                              │
│ Result: OUT_lookupSpecTag[0] = 0b0101010 (tag 42)          │
│         OUT_lookupAvail[0] = 0 (NOT READY) ✗                │
└─────────────────────────────────────────────────────────────┘

┌─────────────────────────────────────────────────────────────┐
│ LOOKUP 1: x5 (rs2)                                          │
├─────────────────────────────────────────────────────────────┤
│ Step 1: specTag[5] = 0b0110101 (tag 53)                    │
│ Step 2: Extract MSB = 0 (normal tag)                        │
│ Step 3: tagAvail[53] = 0 (not ready)                        │
│ Step 4: Check writeback... no match                         │
│ Step 5: Check same-cycle issue... no match                  │
│                                                              │

Conclusion: Instruction must wait in issue queue until tags 42 and 53
            become available (marked in tagAvail bitmap)

## Misprediction Recovery Example

**Branch mispredicts, need to flush instructions 14-15:**
```
┌─────────────────────────────────────────────────────────────┐
│ BEFORE MISPREDICTION                                         │
├─────────────────────────────────────────────────────────────┤
│ comTag[5] = 0b0011001 (tag 25)                              │
│ comTag[7] = 0b0001100 (tag 12)                              │
│ comTag[10] = 0b0000110 (tag 6)                              │
│                                                              │
│ specTag[5] = 0b0110101 (tag 53) ⚡ WRONG PATH               │
│ specTag[7] = 0b0101101 (tag 45)                             │
│ specTag[10] = 0b0110110 (tag 54) ⚡ WRONG PATH              │
└─────────────────────────────────────────────────────────────┘

                   ↓ MISPREDICTION DETECTED ↓

┌─────────────────────────────────────────────────────────────┐
│ CYCLE 1: Reset specTag ← comTag                             │
├─────────────────────────────────────────────────────────────┤
│ specTag[5] = 0b0011001 (tag 25) ✓ RESTORED                  │
│ specTag[7] = 0b0001100 (tag 12) ✓ unchanged                 │
│ specTag[10] = 0b0000110 (tag 6) ✓ RESTORED                  │
│                                                              │
│ Tags 53, 54 released, can be reused                         │
│ Tag 45 kept (instruction 13 was pre-branch, still valid)    │
└─────────────────────────────────────────────────────────────┘

┌─────────────────────────────────────────────────────────────┐
│ CYCLES 2+: Replay uncommitted pre-branch instructions       │
├─────────────────────────────────────────────────────────────┤
│ ROB replays: mul x3, x5, x6 → tag 42 (was uncommitted)     │
│              xor x7, x3, x8 → tag 45 (was uncommitted)      │
│                                                              │
│ Rebuilds correct speculative state without committing       │
└─────────────────────────────────────────────────────────────┘
```
